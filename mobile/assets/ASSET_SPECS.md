# Native Asset Naming

All generated assets live under `/mobile/assets` so they can be reviewed before being copied into native platform resource folders.

## Store Icon

- `app-icon-1024.png`: 1024x1024 PNG, no transparency, dark PromptQuill background.

## Splash Screens

- `splash/splash-2732x2732.png`: universal square source.
- `splash/splash-2048x2732.png`: iPad portrait.
- `splash/splash-1668x2388.png`: iPad Pro portrait.
- `splash/splash-1536x2048.png`: iPad portrait.
- `splash/splash-1242x2688.png`: iPhone large portrait.
- `splash/splash-828x1792.png`: iPhone portrait.
- `splash/splash-1080x1920.png`: Android phone screenshot/splash reference.
- `splash/splash-1200x1920.png`: tablet screenshot/splash reference.

## Android Launcher Icons

- `android/mipmap-mdpi/ic_launcher.png`: 48x48.
- `android/mipmap-hdpi/ic_launcher.png`: 72x72.
- `android/mipmap-xhdpi/ic_launcher.png`: 96x96.
- `android/mipmap-xxhdpi/ic_launcher.png`: 144x144.
- `android/mipmap-xxxhdpi/ic_launcher.png`: 192x192.
- Matching `ic_launcher_round.png`, `ic_launcher_foreground.png`, and `ic_launcher_background.png` files are included for each density.
- `android/adaptive-icon/ic_launcher_foreground.png` and `android/adaptive-icon/ic_launcher_background.png` are the 432x432 master adaptive icon layers.

## iOS App Icon

- `ios/AppIcon.appiconset/Contents.json` maps all iPhone, iPad, and App Store marketing icon files.
