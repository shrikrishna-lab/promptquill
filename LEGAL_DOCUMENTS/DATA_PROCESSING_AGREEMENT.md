**Prompt Quill Inc.** and **User/Customer** ("Controller")

## 1. INTRODUCTION

This Data Processing Agreement ("DPA") supplements the Terms of Service and Privacy Policy and defines the conditions under which Prompt Quill processes personal data subject to the General Data Protection Regulation (GDPR) and other applicable data protection laws.

**Parties:**
- **Data Controller:** Individual user or business (you)
- **Data Processor:** Prompt Quill Inc. (us)

---

## 2. SCOPE AND APPLICATION

### 2.1 Applicability

This DPA applies to:
- All processing of personal data for EU residents
- All processing subject to GDPR
- Processing on behalf of the Controller
- Processing through the Service

### 2.2 Data Categories

**Personal Data Processed:**
- Account information (name, email, account ID)
- Payment information (billing address, transaction history)
- Usage data (clicks, pages visited, time online)
- User-generated content (prompts, generated ideas)
- Communication data (support messages, feedback)
- Device information (IP address, device ID, browser)
- Location data (IP-based, if consented)

### 2.3 Processing Activities

Prompt Quill processes data to:
- Provide the Service and features
- Generate AI-powered content
- Maintain account security
- Process payments
- Communicate with users
- Improve the Service
- Comply with legal obligations
- Protect against fraud

---

## 3. PROCESSOR RESPONSIBILITIES

### 3.1 Processor Obligations

Prompt Quill as Processor agrees to:
- Process data only on documented instructions from Controller
- Ensure persons authorized to process data are under confidentiality
- Implement appropriate technical and organizational measures
- Assist Controller in fulfilling data subject rights requests
- Assist Controller in complying with GDPR obligations
- Delete or return personal data upon Controller request
- Make available information to demonstrate compliance
- Allow audits and inspections by Controller or auditor
- Notify Controller of data breaches without undue delay

### 3.2 Processing Limitations

Prompt Quill will NOT:
- Process personal data for other purposes
- Process more data than necessary
- Process beyond the scope authorized
- Transfer data outside authorized mechanisms
- Use data for commercial purposes
- Share data with unauthorized parties
- Retain data longer than necessary
- Sell personal data

## 5. DATA SUBJECT RIGHTS

### 5.1 Controller Must Enable

As Controller, you must enable us to fulfill rights requests:
- **Right to Access** (Article 15): Provide copy of personal data
- **Right to Rectification** (Article 16): Correct inaccurate data
- **Right to Erasure** (Article 17): Delete data ("right to be forgotten")
- **Right to Restrict** (Article 18): Limit how data is processed
- **Right to Portability** (Article 20): Export data in standard format
- **Right to Object** (Article 21): Object to processing
- **Rights Assessment** (Article 22): Challenge automated decisions

### 5.2 Processor Support

Prompt Quill will:
- Respond to Controller requests for information
- Facilitate data subject requests
- Provide data in standard formats
- Assist with data portability requests
- Support audit trails and compliance

### 5.3 Timeframes

- **Response Time:** 30 days (45 days for complex requests)
- **Verification:** We verify identity before disclosure
- **Fees:** No fees unless requests manifestly unfounded or excessive

## 7. DATA SECURITY AND SAFEGUARDS

### 7.1 Technical Measures

Prompt Quill implements:
- **Encryption:** AES-256 for data at rest
- **Transit Security:** TLS 1.2+ for all communications
- **Access Control:** Role-based access, multi-factor authentication
- **Monitoring:** Intrusion detection, anomaly detection
- **Backups:** Automated daily backups, encrypted
- **Incident Response:** 72-hour breach notification procedure

### 7.2 Organizational Measures

- **Data Protection Officer:** Designated and accessible
- **Training:** Annual data protection training
- **Privacy by Design:** Embedded in development
- **Agreements:** Data processing terms with all staff
- **Audit Logs:** Comprehensive audit trails retained

### 7.3 Security Certifications

Prompt Quill maintains:
- OWASP compliance
- Regular penetration testing
- Vulnerability scanning
- Third-party security audits
- Incident response plan (available upon request)

## 8. DATA BREACH NOTIFICATION

### 8.1 Prompt Quill Obligations

In case of breach:
- Notify Controller without undue delay
- Within 72 hours of discovery (if possible)
- Provide information:
  - Nature of breach
  - Likely consequences
  - Data affected (categories and approximate numbers)
  - Steps taken and proposed remedies
  - Processor contact for more information

### 8.2 Controller Obligations

Upon breach notification, Controller must:
- Assess if notification to data subjects required
- Notify authorities if threshold met
- Monitor for controller responsibilities
- Preserve evidence for investigation

### 8.3 Liability

- Processor liable for damages from negligent breaches
- Processor NOT liable for Controller's failure to comply with GDPR
- Processor NOT liable for data subject statutory rights violations

## 9. DATA RETENTION AND DELETION

### 9.1 Retention Periods

| Data Type | Retention | Reason |
|-----------|-----------|--------|
| Account Data | Until deletion request | Service provision |
| Payment Records | 7 years | Legal/tax requirements |
| Login Logs | 90 days | Security |
| Email Preferences | 2 years | Legal compliance |
| Consent Records | 2 years | GDPR compliance |
| Usage Analytics | 1 year | Service improvement |
| Support Tickets | 2 years | Customer service |

### 9.2 Deletion Process

Upon Controller request for deletion:
1. Account data deleted immediately
2. File storage purged
3. Backups deleted (within 90 days)
4. Payment records kept 7 years (legal requirement)
5. Confirmation email sent

### 9.3 Retention After Termination

After account termination:
- 90-day window for reactivation
- After 90 days: Permanent deletion
- Payment records: Retained 7 years
- Audit trail: Retained 5 years (legal hold)
